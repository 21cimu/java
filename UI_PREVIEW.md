## 界面特点

### 1. 顶部导航栏
- **背景**: 紫色渐变 (#667eea → #764ba2)
- **Logo**: 云图标 + "个人云盘" 文字
- **用户信息**: 右侧显示 "欢迎使用 HDFS 云盘"

### 2. 工具栏
- **背景**: 浅灰色 (#fafafa)
- **按钮**:
  - 上传文件 (蓝色主按钮)
  - 新建文件夹 (白色按钮)
  - 删除 (红色按钮，未选中时禁用)
  - 刷新 (白色按钮)
- **搜索框**: 右对齐，250px 宽

### 3. 面包屑导航
- **背景**: 白色
- **样式**: 蓝色可点击链接，箭头分隔
- **交互**: 点击可快速返回上级目录

### 4. 文件列表
- **表格布局**: 
  - 复选框列 (40px)
  - 文件名列 (50%)
  - 类型列 (25%)
  - 操作列 (25%)
- **图标**: 
  - 📁 文件夹
  - 📄 文件
- **悬停效果**: 行背景变浅灰色
- **点击效果**: 
  - 文件夹名: 进入目录
  - 文件名: 蓝色高亮
  - 下载按钮: 下载文件
  - 删除按钮: 删除确认

### 5. 空状态
当目录为空时显示:
```
     📂
   此文件夹为空
```

### 6. 加载状态
操作执行时显示:
```
  ┌─────────────────┐
  │    🔄 旋转圈     │
  │   加载中...      │
  └─────────────────┘
  半透明黑色遮罩
```

## 响应式设计

### 桌面端 (> 768px)
- 完整布局
- 多列显示
- 悬停效果

### 移动端 (< 768px)
- 堆叠布局
- 隐藏"类型"列
- 触摸优化

## 颜色方案

### 主色调
- 主题色: #2196F3 (蓝色)
- 渐变起: #667eea (紫色)
- 渐变止: #764ba2 (深紫)
- 危险色: #f44336 (红色)

### 中性色
- 背景: #f5f5f5 (浅灰)
- 边框: #e0e0e0 (中灰)
- 文字: #333 (深灰)
- 提示: #999 (浅灰)

### 交互状态
- 悬停: #f9f9f9
- 选中: #e3f2fd
- 禁用: opacity 0.5

## 字体

- 系统字体栈: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Microsoft YaHei', Arial, sans-serif
- 基础大小: 14px
- 标题大小: 24px
- 小字大小: 12px

## 动画效果

1. **按钮悬停**: 0.2s 过渡
2. **加载旋转**: 1s 无限循环
3. **行悬停**: 0.2s 背景色过渡
4. **对话框**: 淡入效果

## 用户体验细节

1. **即时反馈**: 所有操作都有加载提示
2. **确认对话框**: 删除操作需要确认
3. **错误提示**: 操作失败显示具体错误信息
4. **空状态**: 空目录显示友好提示
5. **快捷操作**: 
   - Enter 键提交搜索
   - 全选复选框
   - 面包屑快速导航
